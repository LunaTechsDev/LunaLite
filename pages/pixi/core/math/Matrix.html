class Matrix

package pixi.core.math

@:native("PIXI.Matrix")

Constructor

@:value({ ty : 0, tx : 0, d : 1, c : 0, b : 0, a : 1 })new(a:Float = 1, b:Float = 0, c:Float = 0, d:Float = 1, tx:Float = 0, ty:Float = 0)

The Matrix class is now an object, which makes it a lot faster, here is a representation of it : | a | b | tx| | c | d | ty| | 0 | 0 | 1 |
