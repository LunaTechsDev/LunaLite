typedef AudioFile

package mz.types

import mz.types.RPG

The data class for audio file.

volume:Int

The sound's volume (0..100). The default values are 100 for BGM and ME and 80 for BGS and SE.

pitch:Int

The sound's pitch (50..150). The default value is 100.

pan:Int

The pan.

name:String

The sound file name.
