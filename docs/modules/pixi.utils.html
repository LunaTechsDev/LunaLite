Namespace utils

Generalized convenience utilities for PIXI.

example

// Extend PIXI's internal Event Emitter. class MyEmitter extends PIXI.utils.EventEmitter { constructor() { super(); console.log("Emitter created!"); } }

// Get info on current device console.log(PIXI.utils.isMobile);

// Convert hex color to string console.log(PIXI.utils.hex2string(0xff00ff)); // returns: "#ff00ff"
